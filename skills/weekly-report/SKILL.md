# Weekly Report

Turn a week of activity into a short report the operator can read in a minute.

## When to use

- On a weekly cadence, or before a review.
- Whenever the operator asks "what happened this week?".

## Steps

1. **Gather** the week's completed work, decisions, and parked approvals.
2. **Pull** the headline metrics and how they moved.
3. **Summarize** wins, risks, and what's next — three bullets each.
4. **Keep it short** — one screen, no jargon.

## Output

A dated report: **Highlights**, **Metrics**, **Risks**, **Next week** — each a
few bullets. Save it to the workspace and link it from the conversation.
