# Matter checklist

The standard steps for taking a matter from intake to a filing.

1. **Intake** — capture parties, issue, outcome, and run a conflict check
2. **Research** — pull the controlling law and precedent
3. **Draft** — write the document against our [[Firm standards]]
4. **Compliance** — check the draft against regulation
5. **Approval** — park the filing for the operator's sign-off, then file

See [[Acme services agreement]] for the current run.
