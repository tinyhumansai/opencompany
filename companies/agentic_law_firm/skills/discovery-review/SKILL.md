# Discovery Review

Work through a document production, tagging each item for relevance and
privilege, and leave a log that holds up.

## When to use

- A matter enters discovery and documents must be reviewed.
- A production needs a privilege pass before it goes out.

## Steps

1. **Define** the issues and the responsiveness criteria for this matter.
2. **Batch** the documents and review each for relevance to the issues.
3. **Flag** privileged material and note the basis for each call.
4. **Redact** sensitive content that must not be produced.
5. **Log** every decision so the review is defensible.

## Output

A tagged document set (responsive / privileged / redacted) and a review log
recording each call and its basis. A human approves the production.
