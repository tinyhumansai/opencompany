# LiveOps calendar

The rhythm of running a live game.

1. **Read** — pull engagement and revenue signals each beat
2. **Event** — design the next timed event and its rewards
3. **Offers** — tune monetization against [[Monetization principles]]
4. **Store & growth** — refresh the listing and run acquisition
5. **Launch** — go live behind approval, then measure and report

See [[Summer festival]] for the current run.
