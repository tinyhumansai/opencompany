# LiveOps Event

Design a time-boxed event that gives players a reason to return this week and
keeps the game feeling alive.

## When to use

- The LiveOps calendar calls for the next beat.
- Engagement is dipping and needs a fresh hook.

## Steps

1. **Anchor** the event to a theme and the metric it should move.
2. **Design** the activity loop and the rewards that pull players in.
3. **Set** the duration and the cadence so it feels special, not constant.
4. **Coordinate** offers, store, and community around the event.
5. **Plan** the measurement so you know if it worked.

## Output

An event plan: theme, loop, rewards, schedule, and success metric, following
the [[LiveOps calendar]]. Launches behind the operator's approval.
