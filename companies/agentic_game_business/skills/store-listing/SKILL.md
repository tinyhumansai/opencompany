# Store Listing

Improve the store page so more of the players who see it install — no extra
spend required.

## When to use

- Install conversion is lagging the category.
- A big update or event deserves a refreshed listing.

## Steps

1. **Read** the funnel: impressions, page views, and install rate.
2. **Sharpen** the icon and first screenshots — they carry the decision.
3. **Rewrite** the title and subtitle around the core hook and keywords.
4. **Order** screenshots to tell the game's story in three taps.
5. **Plan** an A/B test on the highest-leverage asset.

## Output

A refreshed listing — icon, screenshots, title, description — and a test plan
for the top asset, following the [[Monetization principles]]. Publish on approval.
