# Strategy Deck

Shape the engagement's findings into a deck that carries a decision, not just
information.

## When to use

- An engagement's analysis is done and needs to land with the client.
- A recommendation needs to survive an executive room.

## Steps

1. **Land** the one message the deck must deliver.
2. **Outline** a storyline where each section earns the next.
3. **Write** action-title slides — the takeaway is the headline.
4. **Support** each claim with one clean exhibit, not five.
5. **Close** with the recommendation and the ask.

## Output

A client-ready deck with action titles, a single storyline, and a clear ask —
built on the [[Engagement standards]]. The operator presents it.
