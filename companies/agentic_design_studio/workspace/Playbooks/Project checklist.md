# Project checklist

The standard steps for a studio project.

1. **Brief** — align on goal, audience, and constraints
2. **Research** — validate the audience and needs
3. **Direction** — set the brand and visual direction
4. **Design** — UI, motion, and illustration against [[Design principles]]
5. **Deliver** — hand the system to the operator for direction sign-off
