# Design Critique

Evaluate a design against the brief and the craft bar, and return changes that
make it measurably better.

## When to use

- A design is ready for a review before it advances.
- Work feels off and you need to name why.

## Steps

1. **Re-read** the brief so the critique judges against the goal.
2. **Assess** hierarchy, clarity, and whether the eye goes where intended.
3. **Check** craft — spacing, type, color, and consistency.
4. **Separate** what works from what to change; keep the good.
5. **Prioritize** the changes by impact on the goal.

## Output

A critique separating strengths from prioritized changes, each specific and
tied to the goal, following the [[Design principles]].
