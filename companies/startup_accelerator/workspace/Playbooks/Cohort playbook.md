# Cohort playbook

The standard steps for running a cohort.

1. **Source** — scout and screen applicants against the [[Admission bar]]
2. **Admit** — the operator makes the admission calls
3. **Program** — match mentors and tailor the curriculum
4. **Coach** — track progress and unblock each startup
5. **Demo day** — prepare pitches, connect investors, support alumni

See [[Spring cohort]] for the current run.
