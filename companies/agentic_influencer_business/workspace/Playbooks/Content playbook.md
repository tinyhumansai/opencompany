# Content playbook

The standard steps for a piece of content.

1. **Trend** — find the angle worth making now
2. **Script** — write the hook and beats in the [[Channel voice]]
3. **Produce** — edit the video and package the thumbnail and title
4. **Publish** — coordinate any sponsor, then post and engage
5. **Measure** — read performance and feed it into the next batch

See [[Beginner cooking series]] for the current run.
